**Batch:** E2 (TFS)

**Assignment:** 3 – AI Chatbot using OpenAI API

---

## 📌 Objective
Build a simple AI-powered chatbot with conversation history,
a clean UI, and bonus PDF question-answering feature.

---

## ✅ Features
- 💬 Chat with AI (GPT-3.5 Turbo)
- 🧠 Conversation history (remembers context)
- 📄 PDF Q&A – upload a PDF and ask questions from it
- 🎛️ Custom system prompt via sidebar
- 🗑️ Clear chat button

---

## 🛠️ Technologies Used
- Python
- OpenAI API
- Streamlit
- PyPDF2
- python-dotenv
